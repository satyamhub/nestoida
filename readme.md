# Nestoida

A rental listing platform for PGs, Flats & Hostels in Noida.

## Features
- Admin login system
- Property CRUD
- Image upload
- Approval workflow
- Search by sector

## Setup Instructions
1. Create MySQL database named `nestoida`
2. Import tables
3. Create `db.php` using db.example.php
4. Run on Apache (LAMP)
